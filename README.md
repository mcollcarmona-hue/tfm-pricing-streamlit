# TFM Pricing Aéreo - Streamlit App

Aplicación Streamlit para visualizar la dificultad predictiva del modelo Top 99% de la Fase 4.

Contenido:
- Mapa por aeropuerto físico de salida.
- Mapa por mercado o ciudad de salida.
- Mapa de rutas aeropuerto-aeropuerto.
- Mapa de rutas mercado-mercado.
- Listas de aeropuertos, mercados y rutas fáciles/difíciles de predecir.
- Texto interpretativo para la memoria.

Caducidad configurada hasta: 28/07/2026

Ejecución local:
pip install -r requirements.txt
streamlit run app.py
